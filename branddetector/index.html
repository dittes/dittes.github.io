Brand Identity Detector

Heuristic inspection, not fake omniscience

Detect brand signals from the page itself, from a fetchable URL, or from uploaded screenshots.

This tool separates high-confidence live page inspection from weak cross-origin URL analysis and from image-based palette extraction. It shows the evidence behind every finding.

Truthfulness rule

A static browser app cannot fully inspect arbitrary third-party pages. CORS, cross-origin stylesheets, tainted canvas rules, and CSP can block access. Blocked steps are shown explicitly.

How it works

  1. Open this app from a hosted static URL.
  2. Drag the bookmarklet to your bookmarks bar.
  3. Open the site you want to inspect.
  4. Click the bookmarklet.
  5. The app opens in a new tab and receives the result via postMessage.
Limitations

Some sites block injected scripts with CSP. That is a real browser restriction, not a bug in the app.

What works and what does not

Static app limitations explained honestly

Works well

  • bookmarklet-based inspection on the actual page context
  • metadata and manifest extraction when a URL is fetchable
  • local screenshot and logo palette analysis
  • exporting tokens and summaries

Weak or blocked by browser rules

  • reading arbitrary cross-origin HTML and CSS without CORS
  • inspecting cross-origin stylesheets from another site
  • drawing third-party images to canvas without tainting
  • exact font identification from screenshots only
  • pages with strict Content Security Policy that block injected scripts
